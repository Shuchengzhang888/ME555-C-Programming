##Test Case##
1. Command line Test Case
(1)Command line is not 5
    ./step3 grid_map.txt obstacle.txt 0 
    Invalid Command Line.
(2)Cannot open file
    ./step3 grid.txt obstacle.txt 0 23
    Invalid File.
(3)The start or goal is not legal
    ./step3 grid_map.txt obstacle.txt 0 24
    ./step3 grid_map.txt obstacle.txt -1 24
    ./step3 grid_map.txt obstacle.txt 3 2
    Invalid Command Line.
(4)Other situation about start or goal
    ./step3 grid_map.txt obstacle.txt 1 23
    1 7 13 20 21 22 23 : 2.94031
    ./step3 grid_map.txt obstacle.txt 1 1
    No Path Exists.

2. Test Case in grid_map.txt
(1) Input file does not has $nodes or $edges
    ./step3 grid_map_321.txt obstacle.txt 0 23
    Wrong Input Format.
(2) Wrong order of $nodes or $edges
    ./step3 grid_map_322.txt obstacle.txt 0 23
    Wrong Input Format.
(3) Node input has letter or does not has two spaces
    ./step3 grid_map_3231.txt obstacle.txt 0 23
    Wrong Input Format.
    ./step3 grid_map_3232.txt obstacle.txt 0 23
    Wrong Input Format.
(4) Node has negative coordinate
    ./step3 grid_map_324.txt obstacle.txt 0 23
    0 7 13 20 21 22 23 : 3.18062
(5) Node index is not consecutive (like 1 2 4 3)
    ./step3 grid_map_325.txt obstacle.txt 0 23
    0 6 13 20 21 22 23 : 3.18062
(6) Edge input has letter or does not has one space
    ./step3 grid_map_3261.txt obstacle.txt 0 23
    Wrong Input Format.
    ./step3 grid_map_3262.txt obstacle.txt 0 23
    Wrong Input Format.
(7) Edge u v is not legal(out of range, smaller than 0)
    ./step3 grid_map_3271.txt obstacle.txt 0 23
    Invalid Input Nodes.
    ./step3 grid_map_3272.txt obstacle.txt 0 23
    Invalid Input Nodes.
